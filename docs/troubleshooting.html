Troubleshooting

Build fails (cmake/g++)

  • Install: sudo apt-get install cmake g++ make
  • Ensure C++17: set(CMAKE_CXX_STANDARD 17) in CMakeLists.txt

Server starts but client cannot connect

  • Check host/port: run both on 127.0.0.1:6380 for local testing
  • Firewall might block ports — try localhost or different port

No data after restart

  • Verify AOF path is writable and not gitignored during tests
  • Check AOF replay logs; malformed lines should be skipped (not crash)

High CPU usage

  • Ensure server reads by newline; avoid busy loops (check recv return)
  • Build in Release (-O2) for realistic perf
