Text Protocol

CommandResponseNotes
PINGPONGHealth check
SET k v...OKValue is rest of line
GET k$ value or (nil)Returns value or nil
DEL kOK or (nil)nil if key absent

Examples

printf "PING\nGET foo\nSET foo bar\nGET foo\n" | ./build/kvclient 127.0.0.1 6380
# PONG
# (nil)
# OK
# $ bar
